## Descrição do projeto

<p align="justify">
    Este projeto é uma plataforma de EAD, em que professores podem postar sua matéria para dar aula e o preço de custo por hora. É visado também poder aproximar os alunos dos professores, criar as conexões entre eles.
</p>

<p>Praticando sempre que possivel o desacoplamento de codigo</p>

### Comandos de inicialização e desenvolvimento

```
    // Codigo para criar o projeto web ReactJS e Typescript
    yarn create react-app web --template typescript

    // Hospeda a aplicação no localhost || tsnd: ts-node-dev == Nodemon
    // Toda aplicação com Typescript, deve ser instalado o mesmo
    tsnd --transpile-only --ignore-watch node_modules --respawn src/server.ts

```
